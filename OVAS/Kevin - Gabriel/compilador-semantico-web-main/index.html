Documentación

MiniLenguaje incluido

La demo usa un minilenguaje compacto para que el análisis sea claro y fácil de seguir.

  • Tipos: entero, decimal, texto, logico.
  • Declaración: entero edad = 20;
  • Asignación: edad = edad + 1;
  • Salida: imprimir(nombre);
  • Condición: si (edad >= 18) { ... }
Interpretación

Cómo leer la salida

Después de ejecutar el análisis puedes revisar distintas vistas complementarias.

  • Errores: muestra línea, tipo de problema y sugerencia.
  • Tabla de símbolos: registra variables, tipos, ámbito y valor.
  • Tokens: lista cada lexema y su categoría.
  • Código intermedio: resume lo que el analizador entendió del programa.
  • Explicación: presenta una traza textual de las decisiones del análisis.

Información del proyecto

¿Por qué se creó?

La finalidad es convertir un tema abstracto en una experiencia visual. En lugar de solo leer teoría, el usuario puede escribir instrucciones, ejecutar el análisis y observar cómo se detectan variables, tipos, errores, tabla de símbolos y código intermedio.

Cómo está publicado el proyecto
