Fase 1: Analizador Léxico

Detalles Académicos del Proyecto

Este objeto de aprendizaje se enfoca en la primera fase de la compilación: el análisis léxico. Su objetivo es convertir un flujo de caracteres del código fuente (en sintaxis simplificada de MiniC) en un flujo de tokens válidos.

Integrantes: Marco & Cristian
Gramática/Temática: Clasificación de palabras reservadas, identificadores, operadores y delimitadores mediante expresiones regulares.
Características UX: Interfaz limpia con tabla de tokens, detección de errores léxicos en tiempo real y coloración sintáctica.
Tecnología: HTML5 nativo, CSS3 puro y lógica en JavaScript cliente estático.

Fase 2: Analizador Sintáctico

Detalles Académicos del Proyecto

Esta OVA valida la estructura jerárquica del código de acuerdo con las reglas de una gramática libre de contexto. Mapea la sintaxis y permite al estudiante visualizar la construcción del árbol sintáctico abstracto (AST).

Integrantes: Fabian & Sofi
Gramática/Temática: Análisis sintáctico descendente/ascendente y representación visual de las dependencias jerárquicas del lenguaje.
Características UX: Generación dinámica del árbol AST animado mediante D3.js, editor de código con panel de consola de errores sintácticos.
Tecnología: JavaScript frontend, D3.js para graficado, API backend en Python (FastAPI).

Fase 3: Analizador Semántico

Detalles Académicos del Proyecto

Valida las restricciones que no pueden expresarse mediante gramáticas libres de contexto, tales como la declaración previa de variables, compatibilidad de tipos en operaciones y la coherencia en el retorno de funciones.

Integrantes: Kevin & Gabriel
Gramática/Temática: Chequeo estático de tipos, construcción de la tabla de símbolos y detección de inconsistencias de lógica computacional.
Características UX: Paneles de advertencias semánticas con marcado de línea de código y visor detallado de tabla de símbolos de variables.
Tecnología: JavaScript frontend, HTML5, CSS3, procesamiento completo en el lado del cliente (estático).

Fase 4: Código de Tres Direcciones (TAC)

Detalles Académicos del Proyecto

Traduce la estructura jerárquica del AST en una secuencia de instrucciones lineales simplificadas (Código de Tres Direcciones) representadas formalmente mediante cuádruplas (operador, argumento 1, argumento 2, resultado).

Integrantes: Islender & Jhoan
Gramática/Temática: Generación de variables temporales, traducción sintáctica de estructuras condicionales (If-Else) y bucles (While).
Características UX: Visualización organizada de cuádruplas interactivas, editor con plantillas de prueba del lenguaje MiniC.
Tecnología: Frontend dinámico en HTML5/JS, motor de compilador API backend en Python (FastAPI).

Fundamentos: Simulador Autómatas (FASS)

Detalles Académicos del Proyecto

FASS es una herramienta visual para el diseño y simulación de Autómatas Finitos Deterministas (AFD) y No Deterministas (AFN), que componen el fundamento matemático en el que se basa el análisis léxico de los compiladores.

Integrantes: Ronald & Johel
Gramática/Temática: Definición de estados iniciales y finales, transiciones por símbolo, transiciones vacías (épsilon) y simulación animada de paso de cadena.
Características UX: Lienzo interactivo para arrastrar y conectar nodos, simulación animada paso a paso, historial e importación/exportación JSON.
Tecnología: Vis.js para grafo interactivo, motor JS local para fallback, API Flask opcional para validaciones complejas.

Acerca de la Plataforma de Compiladores

Portal Académico Integrador de OVAS

Esta plataforma ha sido desarrollada para servir como un entorno académico oficial y centralizado de los Objetos Virtuales de Aprendizaje (OVAs) de la asignatura Teoría de Compiladores de la Universidad Simón Bolívar (Sede Cúcuta).

Cada una de las fases del compilador representa un esfuerzo conjunto de investigación, diseño de software y desarrollo técnico por parte de los estudiantes bajo la tutoría y dirección del docente del curso. La plataforma integra estas fases en un flujo secuencial interactivo que ayuda a los estudiantes de semestres venideros a asimilar de forma visual los conceptos abstractos de la ingeniería de lenguajes de programación.

Objetivos de la plataforma:

  • Centralizar y preservar el software didáctico generado en la universidad.
  • Proveer simuladores interactivos listos para uso pedagógico en aulas de clase.
  • Fomentar la cultura de desarrollo de software, investigación e innovación tecnológica en el programa de Ingeniería de Sistemas.
